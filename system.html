System Features

Scalability

Our system can grow with your workload and tasks automatically.

Security

All communication is encrypted by default and data at rest can be encrypted. Additionally, we support data provenance so you can see the exact path of the data through the system.

Monitoring

The system is built upon the idea of real-time monitoring, alerting, and self-correction. As an operator, you should be able to work without stopping.

Multi-Cloud

Don't put all your eggs in one basket, our system can run across multiple installation scenarios: self-hosted, AWS, Azure, and GCP.

System Overview

Components

Process

The process or data that is going to being analyzed.

Process Collector

An extendable software component that captures data related to the process being performed.

Process Classification Engine

This component converts the raw data into processed information for analysis.

Process Inspection Engine

the core of our system. It performs analysis on the processed data and creates alerts, warnings, and information for the users.

Alarms and Alerts

signals for user and other systems relating to alarms and alerts in the data

Knowledge Expert

an individual, or individuals who understands the process being monitored.

Process Description

an encoding of the process and its steps by the knowledge expert(s).

Raw Process Data DB

a database that holds the raw information of the process.

Processed Data DB

a database that holds the processed information of the process.

Warnings, Alerts, and Insights DB

a database that holds the analysis and information from the data.
