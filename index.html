GitHub Development

Welcome to the GitHub Pages for Athabasca Holdings, LLC Development Team. Our goal is to help companies streamline business processes through the use of Artificial Intelligence, analytics, and Domain Knowledge Experts.

Using an extensible system we can quickly ingest information, rules, and encode domain knowledge to locate inefficiencies, security concerns, and provide remediation steps.

System Overview

The data processing side of the system is a fairly common design. Requiring collectors to convert the processes into data that can be understood by the system.

  • Knowledge Expert - an individual, or individuals who understands the process being monitored.
  • Process Description - an encoding of the process and its steps by the knowledge expert(s).
  • Process - the process being analyzed.
  • Process Collector - the software that captures data related to the process being performed.
  • Process Classification Engine - converts the raw data into processed information for analysis.
  • Process Inspection Engine - the core of our system. It performs analysis on the processed data and creates alerts, warnings, and information for the users.
  • Raw Process Data DB - a database that holds the raw information of the process.
  • Processed Data DB - a database that holds the processed information of the process.
  • Warnings, Alerts, and Insights DB - a database that holds the analysis and information from the data.
  • Alarms and Alerts - signals for user and other systems relating to alarms and alerts in the data
